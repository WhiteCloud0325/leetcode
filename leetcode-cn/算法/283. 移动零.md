## 题目
给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

示例:

输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
说明:

必须在原数组上操作，不能拷贝额外的数组。
尽量减少操作次数。
## 答案

#### 方法一
双指针，需要注意的是当两个指针不在同一位置需要将快指针位置置位0
```
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int res = 0;
        for (int i = 0; i < nums.size(); ++i) {
            if (nums[i] == 0)  continue;
            nums[res++] = nums[i];
            if (res != i + 1)  nums[i] = 0;
        }
        return;
    }
};
```
